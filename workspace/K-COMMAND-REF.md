## 快速参考

### 应用编号安装（最常用）
```bash
k app                     # 浏览应用市场（显示编号列表）
k app 13                  # 直接安装 Cloudreve 网盘（第13项）
k app <编号>              # 通过编号快速安装任意应用
```

## 注意事项

1. **编号不连续**: 部分编号未使用或为预留/重复项，以实际输出为准
2. **安装前确认**: `k app` 可先浏览列表再选择编号
3. **需要Docker**: 大部分应用依赖 Docker 环境运行
4. **网络要求**: 部分工具需要外网访问权限
