# Company Information

_公司信息记录_

## 基本信息

- **公司名称：** 四川里伍铜业股份有限公司（Sichuan Liwu Copper Industry Co., Ltd.）
- **前身：** 四川省里伍铜矿，始建于1988年
- **股份制改造：** 1998年完成，成立四川里伍铜业股份有限公司
- **注册资本：** 22,550万元
- **企业性质：** 国有控股企业（甘孜州国资委实际控制）
- **上市状态：** 曾推进IPO工作（辅导机构进驻），具体上市情况待确认

## 地理布局

- **注册地/总部：** 四川省康定市炉城镇向阳街
- **第二办公区：** 成都
- **主体矿山：** 甘孜藏族自治州九龙县魁多镇、烟袋镇境内（里伍铜矿）
- **外围矿产：** 
  - 黑牛洞中型富铜矿（提交储量报告）
  - 中咀小型富铜矿（开发项目部成立）

## 业务板块

### 核心业务
- **矿业：** 自有铜矿开采及选矿业务，主要产品为铜精矿
- **水电：** 通过控股子公司雅砻江矿业开展三垭河梯级电站开发（已并网发电）
- **勘查：** 全资收购四川金伯利地质勘查有限公司

### 子公司与关联公司
1. 雅砻江矿业公司
2. 里铜电力公司
3. 金伯利地勘公司
4. 丹巴协作铂镍有限责任公司
5. 五家子公司及一家参股商业银行

## 员工概况

- **在册员工：** 870人
- **学历结构：** 大专及以上员工400余人
- **职称分布：** 高级职称188人，中级职称23人，初级职称1人

- 四川省工业企业最佳效益100强
- 甘孜州纳税大户
- 国务院表彰先进企业（具体奖项待确认）

## 重大事件时间线

| 年份 | 事件 |
|------|------|
| 1988年 | 由中国有色金属工业总公司和甘孜州人民政府共同出资筹建里伍铜矿 |
| 1995年9月 | 里伍铜矿日处理原矿500吨的采选生产线正式竣工投产 |
| 1998年 | 完成股份制改造，成立股份有限公司 |
| ~2000s | 进行劳动用工制度改革，全员国有企业身份转换 |
| ~2010s | 九龙县国资公司成为第二大股东；累计投入2亿多元开展找矿工作 |
| ~2015年 | 成立里铜电力有限责任公司 |
| ~2016年 | 联合收购丹巴协作铂镍公司 |
| ~2017年 | 全资收购金伯利地勘公司 |
| ~2018-2019 | 试行集团化管理和管理流程再造；上市辅导机构进驻，IPO前期工作启动 |
| 2019年1月18日 | 公司总部搬迁揭牌仪式在康定举行 |
| 2020年 | 外围找矿提交黑牛洞中型富铜矿储量报告；两矿纳入全国绿色矿山名录 |
