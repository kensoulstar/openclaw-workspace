## 二、组织架构与人员规模

### 组织体系
- 总部下设**十个职能部门**、一座直属矿山和一个项目部
- 旗下拥有五家子公司及一家参股商业银行：
  - 雅砻江矿业公司（控股子公司，被评为"四川省健康企业"）
  - 里铜电力有限责任公司（2007年成立，开展三垭河梯级电站水电开发，已全部并网发电）
  - 金伯利地勘公司（原四川金伯利地质勘查有限公司，2009年全资收购）
  - 丹巴协作铂镍有限责任公司（2008年联合收购）
  - 其他子公司

## 三、公司产业板块

### 1. 矿产资源（核心主业）
- **里伍铜矿**：主体矿山，日处理原矿500吨采选生产线
- **黑牛洞铜矿**：2011年提交中型富铜矿储量报告，1500吨/日采选工程立项核准
- **中咀铜矿**：2013年提交小型富铜矿储量报告

### 2. 水电资源
- 里铜电力公司运营三垭河梯级电站
- 全部机组已并网发电

### 3. 资源勘查
- 累计投入2亿多元在里伍周边开展找矿工作
- 目标：实现"再造一个里伍"
- 铜矿找矿成果完成四川省"十一五"铜矿找矿计划的80%

### 4. 金融业务
- 参股商业银行

## 四、发展历程

| 时间 | 事件 |
|------|------|
| **1988年** | 由中国有色金属工业总公司和甘孜州人民政府共同出资筹建四川省里伍铜矿 |
| **1994年** | 完成基本建设进入试生产阶段 |
| **1995年9月** | 里伍铜矿日处理原矿500吨的采选生产线正式竣工投产 |
| **1998年** | 完成股份制改造，四川里伍铜业股份有限公司成立 |
| **2004年** | 劳动用工制度改革，全员国有企业身份转换 |
| **2006年** | 九龙县龙财矿业矿山资产对公司增资，九龙县国资公司成为第二大股东 |
| **2007年** | 收购成都地质矿产研究所探矿权；成立里铜电力公司 |
| **2008年** | 联合收购丹巴协作铂镍公司 |
| **2009年** | 全资收购四川金伯利地质勘查有限公司 |
| **2010年初** | 试行集团化管理和管理流程再造 |
| **2010年** | 上市辅导机构进驻，上市前期工作启动；总部搬迁康定揭牌 |
| **2011年1月** | 总部搬迁揭牌仪式在康定举行 |
| **2011年** | 提交黑牛洞中型富铜矿储量报告；雅砻江矿业成立 |
| **2013年** | 提交中咀小型富铜矿储量报告；中咀铜矿开发项目部成立 |

- 国务院评为"民族团结进步模范单位"
