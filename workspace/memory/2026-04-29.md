- 从 GitHub 仓库 Agents365-ai/drawio-skill 安装技能到 `~/.openclaw/workspace/skills/drawio-skill/`
- draw.io CLI 在服务器上存在：`/usr/bin/drawio`（版本 29.7.9）
- 技能依赖桌面版 draw.io CLI，不支持 Docker 版
- Gateway 重启后技能生效

### draw.io CLI 调用经验

- 命令格式：`/usr/bin/drawio -x <file.drawio> -f png -o <output.png>`
- 需要 `DISPLAY=:1` 环境变量（Ubuntu 桌面环境）
- XML 里的中文字符必须用 UTF-8 编码，不能用 `&nbsp;` 等 HTML 实体
- draw.io 源文件是 .drawio XML，可用 draw.io 打开编辑
- 导出 SVG 格式可保留矢量，用 Inkscape/Figma/AI 继续编辑
