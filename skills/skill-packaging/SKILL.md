# Skill Workflow Packaging

## Overview

Package reusable workflows as discoverable skills with command metadata, invocation contracts, char-budgeted listings, forked execution, and MCP-sourced skill merging.

## Source Anchors

- Skill system architecture
- Workflow packaging patterns

## Workflow

1. Define skill command metadata
2. Establish invocation contracts
3. Create char-budgeted listings
4. Implement forked execution
5. Support MCP-sourced skill merging

## Design Rules

- Clear command metadata for discoverability
- Explicit invocation contracts
- Character budget awareness
- Fork-safe execution
- MCP integration for skill sourcing

## Output

- Skill packaging specification
- Invocation contract schema
- Character budget guidelines
