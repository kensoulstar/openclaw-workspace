# ClaudeInChrome Skill

自动化 Chrome 浏览器操作。

## 功能概述

通过 Chrome 扩展与浏览器交互：
- 点击页面元素
- 填写表单
- 截图
- 读取控制台日志
- 导航到网站
- 在新标签页中打开页面

## 前提条件

1. 安装 Claude for Chrome 扩展
2. 在扩展设置中配置站点级权限
3. 确保 Chrome 正在运行且扩展已启用

## 激活消息

```
现在可以使用 Chrome 浏览器自动化工具了。
请先调用 tabs_context_mcp 获取当前浏览器标签页信息。
```

## 常用操作

### 1. 获取当前标签页

```
调用: tabs_context_mcp
获取: 当前打开的所有标签页列表
```

### 2. 导航到网站

```
调用: navigate_mcp
参数: url="https://example.com"
```

### 3. 点击元素

```
调用: click_element_mcp
参数: selector="#button-id" 或 "button.submit"
```

### 4. 填写表单

```
调用: fill_form_mcp
参数:
  - selector: "input[name='email']"
  - value: "user@example.com"
```

### 5. 截图

```
调用: screenshot_mcp
参数: full_page=true/false
```

### 6. 读取控制台日志

```
调用: console_logs_mcp
获取: 浏览器控制台输出
```

## 工作流程

1. **获取上下文** - 先调用 `tabs_context_mcp`
2. **执行操作** - 根据需求调用相应工具
3. **验证结果** - 截图或读取状态确认
4. **清理** - 关闭不需要的标签页

## 注意事项

- **权限配置**：首次使用时需要授权站点访问
- **页面加载**：等待页面完全加载后再操作
- **iframe**：处理 iframe 内元素可能需要额外步骤
- **动态内容**：AJAX 加载的内容可能需要等待

## 错误处理

| 错误 | 原因 | 解决 |
|------|------|------|
| 元素未找到 | 选择器错误或页面未加载 | 等待加载或更换选择器 |
| 权限拒绝 | 站点未授权 | 在扩展中授权 |
| 操作超时 | 页面响应慢 | 增加等待时间 |
